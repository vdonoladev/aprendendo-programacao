- URI
    - Conceito
        - Uniform Resource Identifier
            - Identificar um Recurso
            - Nome ou Localização
            - Exemplo
                - Você é um Recurso
                - Seu nome
                - Sua localização
    - Resource
        - O alvo do pedido
        - Qualquer coisa identificável / Entidade
            - Digital
                - Email
                    - mailto
            - Abstrata
                - Sessão
                - Autenticação
            - Física
                - Produtos
                - Usuários
            - Se podemos identificar, nomear, endereças ou manipular, estamos falando de um recurso
    - Locator
    - Name
